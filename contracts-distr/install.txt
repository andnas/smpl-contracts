Требования (проверено на версии)
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
- СУБД MySQL5 (5.6.21)
- Apache Tomcat (7.0.55 без дополнительных библиотек)
- JRE (1.7.0_67-b01, 1.8.0_20-b26)



Последовательность установки
~~~~~~~~~~~~~~~~~~~~~~~~~~~~

1. В СУБД MySQL5 создать новую схему contracts
   CREATE SCHEMA `contracts` ;

2. Выполнить в созданной схеме скрипт make_db.sql

3. В файле contracts.war\WEB-INF\classes\spring-config.xml
   скорректировать пользователя и пароль для доступа к БД (<bean  id="dataSource">)

4. Скопировать contracts.war в папку <CATALINA_HOME>\webapps

5. Зайти по ссылке http://localhost:8080/contracts

6. Нажать на пункт меню "Сгенерировать" (в БД будут созданы 20 компаний)

7. Нажать на пункт меню "Компании". Здесь работает фильтр по 
   организационно-правовым формам организаций.
